A tool that allows you to search for Wikipedia articles with the results being returned sorted by (approximate) popularity.
e.g. Search "monkey" to find Arctic Monkeys, Monkey, 12 Monkeys, Monkepox etc., sorted by total pageviews in the past few years.

Dump taken from https://dumps.wikimedia.org/other/pagetitles/ (chose a day then got enwiki .gz file)
Pageview data added by using Pageviews data API https://wikimedia.org/api/rest_v1/#/
